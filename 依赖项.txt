sh
Copy code

numpy - 主要用于处理大型多维数组和矩阵，以及大量的高级数学函数来操作这些数组。

sh
Copy code

pyautogui - 用于程序控制鼠标和键盘，以自动化GUI操作。

sh
Copy code

pygetwindow - 用于程序控制窗口的大小、位置以及其他属性。

sh
Copy code

keyboard - 允许您全局监听键盘，或发送键盘按键。

Copy code
